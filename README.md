# ProjekNew - Facility Booking & Management System
Sistem manajemen dan pemesanan fasilitas (Facility Booking) beserta pencatatan transaksi yang dibangun menggunakan **Laravel**, **Filament Admin Panel**, dan **Google OAuth**. Sistem ini memiliki fitur untuk pengguna memesan fasilitas melalui antarmuka web, serta menyediakan backend API dan panel admin untuk mengelola data operasional.
## 🚀 Fitur Utama
- **Booking & Transactions**: Halaman untuk pengguna melakukan pemesanan (booking) fasilitas dan melihat daftar transaksi.
- **Filament Admin Panel**: Dashboard admin yang interaktif untuk manajemen data fasilitas, pengguna, transaksi, dan pengaturan aplikasi.
- **Autentikasi Google**: Integrasi Login with Google (OAuth) agar pengguna dapat login dengan aman dan cepat tanpa mendaftar manual.
- **RESTful API**: Endpoint API terstruktur untuk kebutuhan integrasi ke berbagai platform.
- **Nginx Ready**: Terdapat konfigurasi Nginx bawaan untuk kebutuhan deployment ke production.
## 🛠️ Prasyarat (Tech Stack)
Sebelum menjalankan projek ini, pastikan sistem kamu memiliki:
- PHP >= 8.2
- Composer
- Node.js & NPM
- MySQL / PostgreSQL
- Nginx (Opsional, untuk environment server)
